What is Ron Burgundy?

Ron Burgundy is a Software Installation System that can be easily integrated into your new or existing projects.

  • Server Requirements
  • How to Install?
  • Support

Features

  • AJAX interactive
  • Point-and-click Installer
  • Easily integrate into your existing or new projects

Server Requirements

  • MySQL
  • PHP 5.x
  • A webserver (Apache is recommended)

Install

  • Access your target site via your preferred web browser
  • Edit ./includes/config/rbrgvnd.ini
  • A point-and-click Installation interface should appear — follow the on-screen instructions.
